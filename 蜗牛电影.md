POST /api/wxlogin

```yaml
code: string
userInfo: string
```

## POST 用户登录

POST /api/login

> Body 请求参数

```yaml
pwd: "12345"
email: alo2@qq.com
username: string

```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 否 |none|
|» pwd|body|string| 是 |none|
|» email|body|string| 是 |none|
|» username|body|string| 是 |none|

> 返回示例

> 成功

```json

```
